# vue-next-test

使用 Vue CLI V4.x搭建项目完成之后，在项目内执行以下命令项目会自动升级到 Vue3.0 Beta.

```
vue add vue-next
```

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
